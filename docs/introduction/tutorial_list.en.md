## Recommended Learning Path

!!! tip "Suggested Order"
    1. Start with **Core API Tutorials** to learn the basics of Isaac Sim
    2. Move to **Robot Setup** to understand how to work with robot models
    3. Then proceed to specialized tutorials such as **ROS 2**, **Isaac Lab**, **Sensors**, etc.

!!! note "Prerequisites"
    All tutorials assume that Isaac Sim is already installed.
    If you haven't installed it yet, please refer to the [official documentation](https://docs.isaacsim.omniverse.nvidia.com/latest/installation/index.html).
