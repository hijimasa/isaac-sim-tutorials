# センサーチュートリアル

<span class="badge badge-beginner">Beginner</span>

Isaac Sim の RTX センサーや PhysX センサーの使い方チュートリアルです。

## 概要

このチュートリアルシリーズでは、以下の内容を学びます：

- カメラセンサーの設定と画像取得
- LiDAR センサーのシミュレーション
- IMU / Contact センサーの利用
- センサーデータの可視化
