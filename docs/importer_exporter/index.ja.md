# インポート・エクスポートチュートリアル

<span class="badge badge-intermediate">Intermediate</span>

URDF、USD、CAD インポーターおよび URDF エクスポーターを使用してアセットをインポート・エクスポートするチュートリアルです。

## 概要

このチュートリアルシリーズでは、以下の内容を学びます：

- URDF ファイルのインポートと設定
- CAD データ（STEP / IGES）の変換
- USD アセットのエクスポート
- メッシュの最適化とマテリアル設定
