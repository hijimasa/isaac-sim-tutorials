# モバイルロボットのリギング

## 学習目標

このチュートリアルを修了すると、以下の内容を習得できます：

- ロボットアセットのジョイント構成の分析
- USD 階層の整理と XForm によるグループ化
- コリジョン近似の割り当て（Convex Decomposition、シリンダーなど）
- プリズマティック / リボリュートジョイントの追加とドライブ設定
- アーティキュレーションの追加
- アセットの単位変換

## はじめに

### 前提条件

- [チュートリアル 3: 基本ロボットのアーティキュレーション](03_articulate_robot.md) を完了していること

### 所要時間

約 30 分

### 概要

このチュートリアルでは、リギングされていないフォークリフトの USD アセットを、完全にアーティキュレーションされたモバイルロボットに変換します。階層の整理、コリジョンメッシュの割り当て、ジョイントとドライブの定義、アーティキュレーションの追加を行います。

## ジョイントの特定

フォークリフトの7つの自由度を分析します：

- 4つの非駆動ローラー車輪
- 1つのプリズマティック（直動）フォークジョイント
- 1つの車輪回転ジョイント
- 1つのピボットジョイント

## 階層の整理

1. XForm を作成し、メッシュを論理的にグループ化します（ボディ、リフト、車輪）。

2. 適切な親子関係とトランスフォームの整列を設定します。

    ![フォークリフト（トランスフォームなし）](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_1.png)

    ![フォークリフト（トランスフォームあり）](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_2.png)

## コリジョンメッシュの割り当て

1. ボディパーツに Convex Decomposition を適用します。

2. 車輪にシリンダー近似を適用します。

3. 自己貫通を防ぐようコリジョンを設定します。

    ![Convex Decomposition](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_3.png)

    ![シリンダーコリジョン近似](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_5.png)

## ジョイントとドライブの追加

1. フォークの動き用にプリズマティックジョイントを作成します。

2. 車輪用にリボリュートジョイントを作成します。

3. 適切なダンピングとスティフネスパラメータを設定します。

    ![プリズマティックジョイント](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_6.png)

    ![リボリュートジョイント](https://docs.isaacsim.omniverse.nvidia.com/5.1.0/_images/isim_4.5_full_tut_gui_rig_forklift_7.png)

## アーティキュレーションの追加

すべてのジョイントを単一の物理ソルバーチェーンにリンクするため、アーティキュレーションルートを適用します。

## アセット単位の変換

センチメートルベースのアセットをメートルベースのシーンに統合する際の単位変換に対応します。

## まとめ

このチュートリアルでは以下のトピックを扱いました：

1. ロボットアセットの**ジョイント構成**の分析
2. **USD 階層の整理**と XForm によるグループ化

4. **ジョイントとドライブ**の追加と設定
5. **アーティキュレーション**の追加
6. **単位変換**への対応
