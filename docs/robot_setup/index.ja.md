# ロボットセットアップチュートリアル

<span class="badge badge-beginner">Beginner</span>

Isaac Sim でのロボット組み立てとアセット最適化のための入門チュートリアルです。

## 概要

このチュートリアルシリーズでは、以下の内容を学びます：

- ロボットモデル（URDF / USD）の読み込み
- ジョイントの設定とアーティキュレーション
- コリジョンメッシュの最適化
- ロボットの物理パラメータ調整
