## Quick Start

!!! tip "Environment Setup"
    If you haven't installed Isaac Sim yet, please refer to the
    [NVIDIA Official Installation Guide](https://docs.isaacsim.omniverse.nvidia.com/latest/installation/index.html).

!!! info "Prerequisites"
    - Basic knowledge of Python
    - Basic Linux (Ubuntu) operations
    - Basic understanding of ROS 2 (for ROS tutorials)
